기본단계 실행되는 코드 첨부 기본단계.zip

기본단계를 보면 입력파일의 의미와 칼로리 계산 방법이 나와 있습니다.



상속 구현

- food2.txt, eatt2.txt, 상속단계 실행 결과.txt

- printNoLn, printEat, getKcal, getDetail 메소드를 오버라이드함  (Food 상속 클래스)

- 상속 클래스는 분량(ml, cc, g)으로 식사량 표시가 가능함

- 식사량에서 분량으로 또는 개수로 표시한 것을 계산하여 칼로리를 출력한다.

- 상속클래스는 입력에서 분량과 단위를 추가로 읽고 계산에서 분량으로 칼로리 계산이 가능해야 함

  상속 클래스에 대한 칼로리 계산을 출력으로 보여준다

  우유 100ml(62kcal) - 125kcal/1개 -> 125kcal*100/200ml=62kcal

  = 우유가 1개 200ml이고 125kcal인데 100ml를 먹었으니 100/200ml로 계산하면 62kcal가 된다
